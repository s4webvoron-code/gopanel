# Contributing

Спасибо за интерес к проекту GoPanel.

## Как помочь

- Сообщать об ошибках
- Предлагать улучшения
- Отправлять Pull Request

## Требования

- Ubuntu 24.04
- Python 3.12+
- Соблюдение существующего стиля кода

## Pull Requests

Перед отправкой:

1. Проверьте работоспособность изменений.
2. Убедитесь, что не ломается существующий функционал.
3. Опишите внесённые изменения.

## Issues

При создании Issue желательно указать:

- версию GoPanel;
- версию Ubuntu;
- шаги воспроизведения проблемы;
- лог ошибки.
